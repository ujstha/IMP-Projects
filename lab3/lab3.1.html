    • 3.1 Web Sockets

    Nowadays we have very elegant JavaScript APIs we can use to create "Interactive Multimedia Applications". This article was used to answer few questions in this lab set.

  • What kind of problems web sockets solve? In what kind of situations you should use them?

    • Web Sockets solve the problem of the browser of needing to ask for the data from the server in certain interval of time to load the latest information from the server, by making it possible to connect between the web browser and a server. Both sides were able to send and receive data anytime without making any request over and over again.

    Web Sockets can be used in projects where real-time updates are required. Some of the projects where web sockets can be used are:
    Live Game Scores, Multiplayer Games, etc.
