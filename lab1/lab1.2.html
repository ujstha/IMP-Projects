    • 1.2 Coding Convetions

    Coding Convention is a set of rules that is used while coding to make the codes easily readable as well as manageable.

    Down below are few coding conventions that I am going to use during this project:

      HTML
  • Double quoted (" ") syntax on attributes
  • Proper use of: <!DOCTYPE html>
  • Language attribute: <html lang="en-us"></html>
  • Character encoding: <meta charset=UTF-8>
      • CSS
  • Terminate each declarations with semicolon
  • Lower case all hex values and short hex values if any
  • Use of lowercase classes and dashes to separate word
  • Declaration order in positioning, box model, typographic, visual
  • Commented codes for easy readability
