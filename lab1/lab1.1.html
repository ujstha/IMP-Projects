    • 1.1 Short Definitions

  • HTML

    • HyperText Markup Language, HTML in short, is a language that is used to a create webpage. It is the structure of the webpage. It is also considered to be the bone of a webpage.

  • CSS

    • Cascading Style Sheets, CSS in short, is the styling of a webpage. It controls layout of the page like font, color, background, animation, etc.

  • JavaScript

    • JavaScript, JS in short, is an object oriented programming language that is used to change the behaviour of a webpage and make an interactive effects on the webpage.

  • HTTP

    • HyperText Transfer Protocol, HTTP in short, is a set of rules used to transfer files on World Wide Web (WWW). When a url is entered on the browser, it sends a HTTP command to the web-server directing it to fetch the requested webpage on the browser.

  • HTML5

    • The version 5 of HTML is HTML5. Compared to older versions of HTML, HTML5 has few updates on their features and has better way on handling errors. It has one common interface to make loading easier on different browsers.

  • Dynamic HTML

    • Dynamic HyperText Markup Language, DHTML in short, is the term used as a collection of various technologies such as: HTML, CSS, JavaScript to make webpages animated and more interactive.
