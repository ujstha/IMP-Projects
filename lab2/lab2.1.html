    • 2.1 Responsive Web Design

    Responsive web design, RWD in short, is one of the trendiest technology buzz words in web development. The terms was originally introduced in Ethan Marcotteâ€™s A List Apart article in 2010.

  • What is the purpose of the RWD? Why it is used? How things were done before RWD? What is the difference?

    • Responsive Web Design provides the optimal viewing experience of a website, regardless the type of device the user is seeing it on.

    RWD is used to is to resize, hide, shrink, enlarge, or move the content using HTML and CSS and also to make webpages look good on any screen.

    Websites or applications were used to be designed for a 15-inch monitor and incompatibilities between browsers were aside, before RWD came along.

    It has one single layout that always changes according to the size of the screen.

  • How css media queries can help you building responsive web sites?

    • Media Queries (@media) is a CSS technique introduced in CSS3. It extends the functions of media types and allow for more precise display rules in style sheets, only if a certain condition is true.

  • Why to use em (font) based sizing for example margins?

    • If there are many places in the webpage to define its font-size then it will be difficult to remember all the font-sizes so in this case, use of "em" comes in handy.

  • Article is already 7 years old, do we have any new technology which can be used to achieve same type of UI responsiveness as described in the article.

    • Yes, we sure have new technologies for achieving same type of UI responsiveness. Such as: CSS-grid, Flexbox.
