    • 5.1 Getting Familiar with jQuery

    These articles: Article 1 and Article 2 were used to get familiar with jQuery and learn how jQuery works and its benefits.

  • What you can do with the jQuery?

    • jQuery navigates a document. It can also select DOM element and develop ajax applications.

  • How it can help you as web developer?

    • jQuery makes it easier and faster to create abstractions for low-level interaction and animation. It helps in creating web applications.

  • If you think our earlier participant list app, in what kind of issues you could use jQuery?

    • We could use jQuery for selecting input, button elements and to add new elements from text input and use jQuery methods for event handling.

  • What are CDNs? What kind of benefits you get if you use them?

    • Content Delivery Networks, CDNs in short, are used for delivering web contents with high performance and availability.

    It doesn't load framework files from local storage which makes the increment in performance of webpages.
